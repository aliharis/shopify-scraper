# Shopify Scraper

Python based Shopify Scraper based on the scraper originally published on [here](https://www.bestproxyreviews.com/shopify-scraper/).

## Features

* Scraps Products and Reviews.

## Blocks of code

```
app = ShopifyScraper("https://shopnicekicks.com")
app.get_products()
app.print_products()
```
